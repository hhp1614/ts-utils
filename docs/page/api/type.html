    # isArguments

    判断是否为 arguments 对象

    • 参数:
      • value: any 要判断的值
    • 返回值: boolean
    • 例子:
      type.isArguments(function() { return arguments }());
// => true
type.isArguments([1, 2, 3]);
// => false

    # isEmpty

    • 返回值: boolean
    • 例子:

// => false

// => false

// => false

    # isFunction

    判断是否为函数

    • 参数:
      • value: any 要判断的值
    • 返回值: boolean
    • 例子:
      type.isFunction(class Any{});
// => true
type.isFunction(() => {});
// => true
type.isFunction(async () => {});
// => true
type.isFunction(function * Any() {});
// => true
type.isFunction(Math.round);
// => true
type.isFunction(/abc/);
// => false
